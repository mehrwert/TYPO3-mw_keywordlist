Administration
--------------

The installation will add another Menu/Sitemap type to the list of
available content elements. Nothing else has to be done. On large
sites you may notice performance issues when creating the index. See
the Known Problems section.

Installation doesn't require any additional fields in the database.
You can leave you cache untouched, clearing won't be necessary.
